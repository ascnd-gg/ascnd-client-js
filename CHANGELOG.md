# @ascnd-gg/client

## 1.0.0

### Major Changes

- Initial release of `@ascnd-gg/client`
- Migrated from `@ascnd/client` to new GitHub organization
- Full TypeScript support with zero runtime dependencies
- Support for Node.js 18+, browsers, and edge runtimes

### Features

- `submitScore()` - Submit player scores to leaderboards
- `getLeaderboard()` - Retrieve top scores with pagination
- `getPlayerRank()` - Get a specific player's rank and percentile
- Automatic request timeout handling
- Custom fetch implementation support
- Comprehensive TypeScript types
